# Amazon-Sales-Dashboard-PowerBI

## Project Overview

This project is an interactive Power BI dashboard built using the Amazon Sales Dataset. The dashboard provides insights into sales performance, profit, shipping costs, market distribution, country-wise sales, city-wise sales, and category analysis.

The goal of this project is to transform raw sales data into meaningful business insights using Power BI visualizations and data modeling techniques.

## Business Insights

- Standard Class contributes the highest sales among shipping modes.
- United States generates the highest sales among all countries.
- New York City is the top-performing city.
- Technology category contributes the largest share of sales.
- APAC and EU markets contribute significantly to overall revenue.
